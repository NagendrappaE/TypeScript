# create project steps

1. create  project folder
mkdir new-typescript-palyground

2. npm init -y
3.npx tsc --init

4. create src folder 

5. create file with <fileName>.ts

6. to compile code from ts to js
 tsc src/fileName>.ts

7.to run the ts file

node src/fileName>.js
